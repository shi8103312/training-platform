## 系统登录

### 登录地址
- 访问 `http://localhost:5173/hr` 或点击前端首页的"管理入口"

### 登录账号
- **默认 HR 管理员账号**: `admin` / `admin123`
- **超级管理员账号**: `super001` / `super123`（拥有所有HR权限）

### 登录流程
1. 输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后自动跳转到 HR 工作台

## 工作台概览

登录后首先看到的是工作台首页，展示以下关键数据：

- **进行中培训**：当前正在进行的培训项目数量
- **参与员工**：已报名参加培训的员工数量
- **待完成人数**：尚未完成培训的学员数量
- **平均完成率**：所有培训项目的平均完成百分比

### 快捷操作
工作台提供四个快捷入口：
- 📚 **创建培训**：创建新的培训项目
- ✏️ **管理项目**：查看和管理所有培训项目
- 📤 **导入部门**：批量导入部门信息
- ✉️ **发送通知**：向员工发送通知

### 学习趋势图
显示近7天的学习参与情况柱状图。

## 培训项目管理

### 创建培训项目

1. 点击"创建培训"或进入"培训管理"页面
2. 点击"新建培训"按钮
3. 填写培训信息：
   - **培训标题**：输入培训项目名称
   - **培训描述**：详细描述培训内容
   - **是否必修**：选择"必修"或"选修"
   - **截止日期**：设置培训的完成截止时间
   - **推送范围**：选择推送范围（全员/指定部门/指定人员）

4. 点击"保存"或"保存并发布"

### 管理培训项目

在"培训管理"页面可以：
- 查看所有培训项目列表
- 按状态筛选（全部/草稿/已发布/已下架/已结束）
- 搜索培训项目
- 编辑培训项目信息
- 发布/下架培训项目
- 删除培训项目

### 培训项目状态

| 状态 | 说明 |
|------|------|
| 草稿 | 刚创建，未发布 |
| 已发布 | 员工可见，可以开始学习 |
| 已下架 | 暂时不可见，但未删除 |
| 已结束 | 超过截止日期 |

### 设置推送范围

创建培训时可以设置哪些员工可以看到：
- **全员**：所有员工都可见
- **指定部门**：选择特定部门，只有该部门员工可见
- **指定人员**：选择具体员工，只有被选中的员工可见

## 材料管理

每个培训项目可以添加多个学习材料（视频或文档）。

### 上传材料

1. 进入培训项目详情
2. 点击"材料管理"标签
3. 点击"上传材料"按钮
4. 填写材料信息：
   - **材料标题**：材料名称
   - **材料类型**：视频/文档
   - **是否必修**：必修材料完成后才能参加考试
   - **排序**：材料显示顺序
5. 选择要上传的文件
6. 点击"上传"

### 材料要求

**视频格式支持**：
- MP4、AVI、MOV、WMV、FLV、WebM
- 最大文件大小：500MB

**文档格式支持**：
- PDF、Word、Excel、PPT
- 最大文件大小：50MB

### 注意事项
- 必修材料：员工必须完成才能参加考试
- 选修材料：完成与否不影响考试资格
- 已上传的视频会自动进行转码处理

## 考试管理

### 创建考试

1. 进入培训项目
2. 点击"考试管理"标签
3. 点击"创建考试"按钮
4. 填写考试信息：
   - **考试标题**：考试名称
   - **考试说明**：考试规则描述
   - **及格分数**：及格分数线（默认60分）
   - **考试时长**：允许的考试时长（分钟）
   - **考试次数限制**：允许的考试次数（0=不限制）
   - **随机出题**：是否打乱题目顺序
   - **显示答案**：考试后是否显示正确答案

5. 添加考题：
   - **单选题**：只有一个正确答案
   - **多选题**：有多个正确答案
   - **判断题**：正确/错误
   - **简答题**：需要人工阅卷

6. 点击"保存"

### 考试规则

- 员工必须完成所有**必修材料**才能参加考试
- 考试过程中切换浏览器标签页会被记录为违规
- 考试时间结束会自动提交
- 考试通过后自动更新学习进度为100%

### 查看考试成绩

在培训项目的"考试管理"中可以看到：
- 各员工的最高成绩
- 已考次数/剩余次数
- 是否通过

## 部门管理

### 添加部门

1. 进入"部门管理"页面
2. 点击"添加部门"按钮
3. 填写部门信息：
   - **部门名称**：部门全称
   - **上级部门**：选择上级部门（可选，用于构建部门树）
4. 点击"保存"

### 批量导入部门

支持 Excel 批量导入：
1. 点击"导入部门"按钮
2. 下载导入模板
3. 填写部门信息（部门名称、上级部门）
4. 上传填写好的 Excel 文件
5. 点击"确认导入"

### 部门结构

系统支持多级部门结构，上级部门可以看到下级部门员工的学习情况。

## 员工管理

### 查看员工列表

在"员工管理"页面可以：
- 查看所有员工账号
- 按部门筛选员工
- 搜索员工（用户名/姓名）
- 编辑员工信息
- 禁用/启用员工账号

### 创建员工账号

1. 点击"添加员工"按钮
2. 填写员工信息：
   - **用户名**：登录账号（唯一）
   - **真实姓名**：员工姓名
   - **邮箱**：员工邮箱
   - **手机号**：联系方式（可选）
   - **所属部门**：选择员工所在部门
3. 点击"保存"

### 员工角色

| 角色 | 说明 |
|------|------|
| 员工 | 普通员工，只能学习和参加考试 |
| HR管理员 | 可以管理培训、部门、员工等 |

## 学习进度查看

### 查看项目整体进度

1. 进入"进度查看"页面
2. 选择培训项目
3. 查看各项统计：
   - **完成率饼图**：已完成/进行中/未开始比例
   - **趋势图**：近7天学习趋势
   - **员工列表**：每位员工的学习状态

### 导出学习报告

1. 选择培训项目和部门
2. 点击"导出报告"按钮
3. 系统生成 Excel 格式的学习进度报告

### 进度计算规则

- **视频完成**：观看进度达到95%以上视为完成
- **文档完成**：标记为已读即完成
- **项目完成**：所有必修材料都完成
- **考试通过**：得分达到及格分数线

## 发送通知

### 发送站内通知

1. 进入"发送通知"页面
2. 填写通知内容：
   - **选择培训项目**：关联的培训项目
   - **通知标题**：通知主题
   - **通知内容**：详细通知内容
3. 设置推送范围：
   - **全员**：发送给所有员工
   - **指定部门**：选择部门
   - **指定人员**：选择具体员工
4. 设置发送时间：
   - **立即发送**：通知立即发出
   - **定时发送**：设置具体发送时间
5. 点击"发送"

### 查看发送历史

页面下方显示历史发送记录，包括：
- 通知标题
- 关联培训项目
- 发送范围
- 发送时间
- 发送人数
- 发送状态

支持导出发送记录为 Excel 文件。

## 系统设置

### 基本设置

- **平台名称**：设置平台显示名称

- 支持的视频格式
- 最大文件大小限制
- 是否启用视频转码
- 是否启用视频加密

- 启用/禁用各类型通知

## 主题切换

系统支持5种主题皮肤切换：

| 主题 | 颜色风格 |
|------|----------|
| 默认蓝紫 | 蓝紫渐变 |
| 天空蓝 | 蓝绿渐变 |
| 森林绿 | 绿色渐变 |
| 日落橙 | 橙色渐变 |
| 浆果红 | 红粉渐变 |

### 切换方式

1. 点击页面顶部右侧的主题选择器
2. 选择喜欢的主题颜色
3. 主题自动切换并保存

### 主题持久化

选择的主题会保存到用户账号中，换设备登录后也会保持相同主题。

## 常见问题

### Q: 员工无法参加考试
A: 检查员工是否完成了所有必修材料，只有完成必修材料才能参加考试。

### Q: 视频上传失败
A: 检查文件格式和大小是否符合要求，以及视频转码服务是否正常运行。

### Q: 忘记密码怎么办
A: 在登录页面点击"忘记密码"，输入用户名和注册邮箱，重置密码链接会发送到邮箱。

### Q: 如何添加超级管理员
A: 需要在数据库中直接插入账号，将 role 字段设置为 0。
